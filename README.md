# Module 3 — Core Skills Drill: SQL Fundamentals

Write three SQL query functions in Python against the provided `drill.db` SQLite database.

## Setup

```bash
pip install -r requirements.txt
```

## Tasks

Create `drill_queries.py` with three functions:

1. `top_departments(db_path)` — Top 3 departments by total salary (JOIN + GROUP BY)
2. `employees_with_projects(db_path)` — All employee-project pairs (3-table JOIN)
3. `salary_rank_by_department(db_path)` — Salary rank within each department (window function)

See the drills page for full specifications.

## Submit

1. Create branch `drill-3-sql-fundamentals`
2. Add `drill_queries.py`
3. Push and open a PR to `main`
4. Paste your PR URL into TalentLMS
